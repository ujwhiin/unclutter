## 三个功能，分别能干什么？

### 文件暂存（Files Hub）

你在整理文件夹时，有没有遇到"我要把这个文件从 A 移到 B，但 B 文件夹在另一个窗口"的情况？以前的做法可能是开两个 Finder 窗口拼在一起。现在更简单：把文件拖到屏幕顶部，Unclutter 自动弹出来接住它，你再去 B 文件夹拖出去就行。

这个"临时停靠站"的用法特别省心，尤其在多任务切换的时候。

### 剪贴板管理（Clipboard History）

Mac 系统自带的剪贴板只记录最后一次复制的内容，之前的全没了。Unclutter 可以记录你复制过的文字和图片，支持最多 50 条历史，重启也不会清空。更实用的是，你可以在剪贴板里直接**编辑文本内容**再粘贴出去——比如存一个邮件模板，每次改几个字就发出去，不用重新打。

还有一个隐私保护功能：可以指定某些 App（比如 1Password）的内容**不被记录**。这个细节做得挺到位的。

### 快速笔记（Quick Notes）

不需要打开备忘录、也不需要打开 Notion，鼠标往上一拖，直接在 Unclutter 的笔记区开始写。支持多条笔记、全文搜索。如果你用 iCloud 或 Dropbox，还能跨设备自动同步。
